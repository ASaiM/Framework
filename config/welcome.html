ASaiM (Auvergne Sequence analysis of intestinal Microbiota) is an framework to analyze microbiota and environmental samples. Read more about ASaiM context

This open-source opiniated framework is based on a custom Galaxy instance. It integrates numerous tools for microbiota and environmental data processing, taxonomic analyses and functional analyses. Several standard but customizable workflows are also proposed to help the analysis of microbial data

For more information about the tools, workflows and ASaiM in general, please read the online documentation. You can also follow our Tutorial to learn how to use the ASaiM framework to analyze microbiota data.

Galaxy is an open platform for supporting data intensive research. Galaxy is developed by The Galaxy Team with the support of many contributors.
